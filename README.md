# Virtual Pet simulation

## Overview
This project is a Virtual Pet Care System that allows users to adopt and interact with a pet by managing its needs such as hunger, thirst, health, and happiness.

## Features
- User account management
- Pet adoption
- Pet interaction (feed, play, groom, etc.)
- Pet status tracking
- Notification system
